# Cortex Roadmap

## Phase 1 — Foundation
- Android project structure
- Compose UI shell
- Room database
- Secure settings
- Capture screen
- Local memory store
- Encrypted JSON export/import

## Phase 2 — Memory and graph
- Entity extraction
- Relation extraction
- Graph persistence
- Search
- Deduplication
- Importance scoring

## Phase 3 — Guidance
- Daily brief
- Thinking mode
- Follow-up questions
- Next-step suggestions
- Pattern discovery

## Phase 4 — OpenRouter
- Model configuration
- Prompt templates
- Structured JSON outputs
- Summarization
- High-level synthesis only

## Phase 5 — Connectors
- Connector SDK
- Configuration UI
- Sync jobs
- Import pipelines
- Provenance display

## Phase 6 — Polish
- Graph visualization improvements
- Better search UX
- Fast startup
- Offline robustness
- Tests and release hardening

## Prioritization rule
Always build the foundation before the intelligence layer, and the intelligence layer before the connector expansion.
